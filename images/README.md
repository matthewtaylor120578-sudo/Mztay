# Microsite assets

Drop these files here for the Dyno Nobel microsite to render correctly:

- `pc-team.jpg` — P&C Partners team photo (hero background, ~2000px wide)
- `dyno-nobel-logo-white.png` — Dyno Nobel logo, white version on transparent bg (for hero band)
- `dyno-nobel-logo.png` — Dyno Nobel logo, navy on transparent bg (for body sections / footer on light bg)

File names are case-sensitive and must match exactly.

If a file is missing the page still loads — the hero falls back to a navy gradient and the client logo falls back to styled text.
